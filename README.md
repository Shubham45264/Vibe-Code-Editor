# 🎨 VibeCode Editor

### *Code. Create. Vibe.*

VibeCode Editor is not just another online code editor — it’s a complete **developer workspace in your browser**.

Designed for speed, simplicity, and intelligence, VibeCode blends the power of modern tooling with AI-driven assistance so you can focus on what really matters: **building great software**.

---

## ✨ Why VibeCode?

* ⚡ No setup required — start coding instantly
* 🧠 AI that understands your code
* 🌐 Full runtime in browser
* 🎯 Clean & distraction-free UI

---

## 🚀 Features

### 💻 Intelligent Playground

* Monaco Editor (VS Code engine)
* WebContainers (Run Node.js in browser)
* File Explorer (Real IDE feel)
* Auto Save & Sync

---

### 🤖 AI Assistant

* Context-aware chat
* Multi-model support (GPT-OSS, CodeLlama)
* Smart debugging & optimization
* Markdown + LaTeX support

---

### 🛠️ Developer Tools

* Integrated Terminal
* Project Dashboard
* Secure Authentication

## 🧱 Tech Stack

* **Framework**: Next.js 15+
* **Editor**: Monaco Editor
* **Runtime**: WebContainers
* **Database**: Prisma + PostgreSQL/MySQL
* **Auth**: NextAuth
* **UI**: Tailwind + ShadCN + Radix
* **Terminal**: xterm.js

---

## ⚡ Getting Started

```bash
git clone https://github.com/Shubham45264/Vibe-Code-Editor.git
cd Vibe-Code-Editor
npm install
```

---

### 🔐 Environment Setup

```env
DATABASE_URL="..."
AUTH_SECRET="..."
GITHUB_ID="..."
GITHUB_SECRET="..."
```

---

### 🗄️ Database

```bash
npx prisma generate
npx prisma db push
```

---

### ▶️ Run Project

```bash
npm run dev
```

👉 Open [http://localhost:3000](http://localhost:3000)
